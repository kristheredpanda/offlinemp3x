# OfflineMP3X
Play your favourite tunes without internet.

OfflineMP3X v1.0.0 Release Notes
Right now it's a basic MP3 player, it features:

• AutoPlay, to automatically move onto the next song.
• Light & Dark Themes, Dark being the default.
• Choose any folder to load songs from, as long as it contains .mp3 files.
• Opt-in/Opt-out option to check for any updates upon opening the program.
